Vítejte u projektu PROJECT-2. Jedná se o plně funkční a interaktivní aplikaci pro správu úkolů (To-Do List). Tento projekt byl navržen s důrazem na uživatelskou přívětivost (UX), čistý kód a efektivní manipulaci s prvky DOM v čistém JavaScriptu.


Key Features:

Task Management: Quickly dd new tasks using the Enter key or button.
Interactive States: Ability to mark a task as completed (cross it out) or permanently delete it.
Responsive Design: The application is fully usable on mobile devices and desktops.


Design & UX:

Before coding, I focused on:
Clearness: A minimalist interface that doesn't distract from the task at hand.
Intuitiveness: Controls that the user can understand in a second.

Technical development:

The application demonstrates techniques for working with pure JavaScript (Vanilla JS) without using external libraries:
Semantic markup and form structure.
Flexbox for element alignment and smooth state styling.
DOM manipulation, Event Listeners and application logic.
